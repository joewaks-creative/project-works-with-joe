# Pilot Program Plan

## Target Group
Market women in Accra, Ghana - approximately 20 participants aged 30-60.

## Duration
4 weeks, starting with an in-person launch.

## Week 1: Launch & Onboarding
- In-person session at a local market.
- Introduce the platform and Adinkra symbols.
- Help participants install the PWA or access via Telegram.
- Teach basic navigation.

## Week 2: Module 1 - Cybersecurity
- Guided lessons on Gye Nyame.
- Daily check-ins via WhatsApp group.
- Practice setting PINs and passwords.

## Week 3: Module 2 - Data Backup
- Lessons on Sankofa.
- Hands-on backup exercises.
- Troubleshooting support.

## Week 4: Review & Feedback
- Complete remaining modules.
- Collect feedback through surveys.
- Graduation ceremony with certificates.

## Metrics
- Completion rates
- Pre/post knowledge assessments
- User satisfaction scores
