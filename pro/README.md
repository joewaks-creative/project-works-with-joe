# Digital Heritage: Ghana Digital Literacy Platform

A mobile-first Progressive Web App using Adinkra symbols to teach digital literacy to Ghanaian audiences.

## Features
- Low-data design for accessibility
- Cultural relevance through Adinkra symbols
- Offline-capable PWA
- Simple, intuitive interface

## Curriculum Development
Lessons developed in collaboration with local Adinkra expert [Name], who provided cultural context and symbol interpretations.

## Prototype
This repository contains a working prototype of the Cybersecurity module using Gye Nyame symbol.

## Tech Stack
- HTML/CSS/JavaScript
- Progressive Web App (PWA)
- Service Worker for offline functionality

## Installation
1. Clone the repository
2. Open index.html in a modern browser
3. Install as PWA on mobile devices
